# 10. Übersicht der Fenster und Navigation

Die folgende Übersicht zeigt alle Fenster der Anwendung und wie diese erreichbar sind:

- **[Manuell]** → Dateninput
  - Gruppentabelle (links)
  - Vereinstabelle (Mitte)
  - Mannschaftstabelle (rechts)
- **[Aus Click-TT]** → click-tt Import
  - Gruppen (oben): Gruppenliste (links), Teams der Gruppe (rechts)
  - Terminmeldung (unten): Vereinsliste (links), Teams des Vereins (rechts)
- **[Aus Datei]** → Dateiauswahldialog
- **[Generieren]** → Bitte Warten
  - Bei Konflikten → Alternativen
- **[Speichern]** → Dateiauswahldialog
- **[Sonstiges]** → Sonstiges
  - Backup laden
  - Ergebnisse exportieren
  - Terminmeldung speichern
  - Konflikte neu auflösen
  - Generator-Tests / Tests aus Datei
  - Konfiguration exportieren/importieren
  - CSV exportieren/importieren (historisch)
  - Abbrechen
- **[Vereinssicht]** → Liste mit Teams des Vereins
- **[Gruppensicht]** → Liste mit Teams der Gruppe
- **[Rechtsklick auf Team]** → Zusatz (Spielwochen-/Spieltagseinstellung)
