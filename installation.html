KeyGenerator – Schlüsselzahlen-Generierungstool

Installation

Voraussetzungen

Für die Installation und den Betrieb des KeyGenerators wird benötigt:

  • Windows 10 oder neuer (64-Bit)
  • .NET 8.0 Desktop Runtime oder neuer

Falls .NET 8.0 noch nicht installiert ist, wird es während der Installation des KeyGenerators automatisch nachgeladen.

Installationsschritte

Den Installer kann hier heruntergeladen werden:

KeyGenerator Installer herunterladen
  1. 1 Die heruntergeladene Datei KeyGeneratorInstaller.msi ausführen.
  2. 2 Den Anweisungen des Installationsassistenten folgen und einen Installationsordner auswählen.
  3. 3 Nach abgeschlossener Installation kann der KeyGenerator über das Startmenü oder die erzeugte Desktop-Verknüpfung gestartet werden.

Hinweis: Windows SmartScreen

Wichtig – Testversion: Der KeyGenerator ist aktuell eine Testversion und daher nicht mit einem kostenpflichtigen Zertifikat signiert. Windows SmartScreen stuft die Quelle der Installationsdatei deshalb als unbekannt ein und zeigt eine Sicherheitswarnung an. Dies ist zu erwarten und kein Anzeichen für Schadsoftware.

Beim Ausführen des Installers erscheint möglicherweise folgendes Fenster:

„Windows hat den PC geschützt"
Der Start von KeyGeneratorInstaller.msi wurde verhindert, da die App von einem unbekannten Herausgeber stammt.

Vorgehen zum Fortfahren:

  1. Im Warnfenster auf „Weitere Informationen" klicken.
  2. Anschließend auf „Trotzdem ausführen" klicken.
  3. Die Installation läuft danach normal weiter.
Hinweis: Dieselbe Warnung kann auch beim ersten Start der Anwendung erscheinen. Auch dort kann sie mit „Weitere Informationen" → „Trotzdem ausführen" übergangen werden.
